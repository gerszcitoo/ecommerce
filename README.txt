El perfil mostrado arriba a la derecha se genera mediante la API "Random User Generator" (https://randomuser.me/) y se guarda en el sessionStorage, para mostrar mejor el uso de la misma.
Los productos creados por el Administrador y el carrito de compras se almacenan en el localStorage para un funcionamiento óptimo y un uso más cómodo y realista de la apliación.

Para iniciar sesión como administrador:
    1. Dirigirse a la página "Administrar".
    2. Ingresar "admin" como usuario, y "123" como contraseña.

Para añadir productos:
    1. Se debe ser administrador.
    2. Dirigirse a "Tienda".
    3. Pulsar el botón "Crear un producto" y completar la información solicitada.
    4. Pulsar el botón "Añadir".
